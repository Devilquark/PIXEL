# Agents PIXEL

Écosystème d'agents spécialisés pour piloter la marque **PIXEL** (vêtement, cible jeune / grand public).

## À quoi ça sert ?

Chaque fiche agent est un **brief réutilisable** : un rôle clair, une mission précise, un ton, des règles. Tu peux les utiliser :
- comme système-prompt quand tu invoques un sous-agent dans Cowork ;
- comme cadre de référence quand tu travailles toi-même sur une de ces missions ;
- comme document de transfert si tu travailles avec une agence externe.

## Les 5 agents

| # | Agent | Mission en 1 phrase |
|---|-------|---------------------|
| 1 | **Brand Guardian** | Vérifie que tout contenu produit respecte l'ADN PIXEL |
| 2 | **Social Content** | Produit du contenu TikTok / Insta / Snap qui parle à la Gen Z |
| 3 | **Drop Manager** | Orchestre le lancement d'une nouvelle collection capsule |
| 4 | **Trend Scout** | Surveille les tendances streetwear et propose des angles |
| 5 | **Influence** | Identifie, brief et suit les micro-influenceurs |

## Statut actuel

**Phase 1 — En attente de fondations.** Ces agents sont des squelettes : ils ont la structure et la mission, mais les sections « Codes de marque » et « Ton de voix » sont à compléter une fois qu'on aura défini :
- positionnement de PIXEL
- persona cible précis
- territoire de marque
- codes visuels (palette, typo, univers)
- charte éditoriale

Une fois ces fondations posées, chaque agent sera enrichi avec les éléments concrets de la marque.

## Comment utiliser un agent

1. Ouvre la fiche de l'agent concerné.
2. Copie le bloc « Brief » dans une nouvelle conversation Claude (ou utilise-le comme prompt système).
3. Fournis le contexte spécifique de ta demande (ex : « on lance une capsule été, voici les 3 pièces »).
4. L'agent te renverra un output formaté selon les règles définies dans sa fiche.
